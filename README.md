## 🚀 快速开始

### 1. 安装Tampermonkey

- **Chrome/Edge**: 从Chrome Web Store安装
- **Firefox**: 从Firefox Add-ons安装

### 2. 安装脚本

1. 复制 `security-tester-v2.0.user.js` 全部内容
2. Tampermonkey面板 → "+" 新建脚本
3. 粘贴并保存 (`Ctrl+S`)

### 3. 运行测试

1. 访问 `https://scauzj.leykeji.com`
2. 登录测试账号
3. 右上角勾选 "启用测试工具"
4. 点击 "运行漏洞验证"
5. 查看报告并导出

## 📊 测试报告

工具会自动生成详细报告,包括:

- ✅ 风险评分(百分比+等级)
- ✅ 漏洞统计(按严重程度分类)
- ✅ 每个漏洞的详细证据
- ✅ 网络请求记录
- ✅ 运行时长统计

**报告导出**: 点击"导出报告"按钮保存为 `.txt` 文件
