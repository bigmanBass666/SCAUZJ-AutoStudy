## 发现的问题与修复

### BUG-001: SPA环境检测失败

**症状**: URL变化后detectEnvironment返回null

**根因**: 新页面video.readyState=0，文本匹配不到时长

**已尝试修复**:
- [x] 延长重试次数到30次（9秒）
- [x] 改进时长提取算法（从后往前找非零HH:MM）
- [ ] 添加DOM readyState检测
- [ ] 添加MutationObserver等待video元素就绪

### BUG-002: autoNext跳转方式

**尝试方案**:
1. [x] 侧边栏链接点击 → 失败（链接可能是动态生成）
2. [x] location.href赋值 → 部分触发，但监听器未捕获
3. [x] location.assign() → 同2

**原因分析**: URL变化监听器在autoNext执行时可能已经失效（因为bot已被清理）
