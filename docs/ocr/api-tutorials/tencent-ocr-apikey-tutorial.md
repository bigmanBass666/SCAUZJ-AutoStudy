# 腾讯云 文字识别 OCR：如何免费领取 API Key（面向普通用户）

---

**适用对象**：非技术背景的普通用户、个人开发者、学生
**预计耗时**：约 15-30 分钟
**所需材料**：手机号（中国大陆）、身份证（用于实名认证）

## 一、这是什么服务

腾讯云文字识别（OCR）是腾讯云推出的一项云端文字识别服务，能够将图片、PDF中的文字内容智能识别并提取为可编辑文本。该服务覆盖通用印刷体识别、手写体识别、身份证识别、银行卡识别、驾驶证识别、票据识别等多种场景，识别准确率高，响应速度快。

**免费额度说明**：腾讯云为开通文字识别服务的用户提供**每月 1,000 次免费调用额度**。这个额度是"共享资源包"形式，适用于通用文字识别、卡证文字识别、票据单据识别等多个接口类型。免费额度每月自动刷新，无需手动领取。

**适合谁用**：适合需要进行图片文字提取、证件识别、票据处理等场景的用户。腾讯云OCR在国内网络环境下访问稳定，接口响应速度快，非常适合国内用户的各类应用场景。

## 二、开始之前需要准备什么

### 必须准备

1. **手机号**：需要一个中国大陆手机号，用于注册腾讯云账号并接收验证码。

2. **身份证**：需要进行实名认证，个人用户需要填写身份证号码并验证。这是开通云服务的必要条件。

### 可选准备

3. **邮箱**：可以绑定邮箱用于接收通知，但不是必须的。

4. **微信**：支持微信扫码登录，方便快捷。

### 关于银行卡

**不需要绑定银行卡**：腾讯云的文字识别服务在使用免费额度期间无需绑定银行卡或充值。只有当您超出免费额度并需要付费使用时，才需要进行充值或绑定支付方式。

### 认证类型说明

腾讯云支持两种实名认证方式：
- **个人认证**：适合个人用户，使用身份证完成认证，流程简单快捷。**推荐普通用户选择此方式。**
- **企业认证**：适合企业用户，需要营业执照等企业资质。

**重要提示**：无论个人认证还是企业认证，免费额度都是每月 1,000 次。因此对于普通用户，个人认证完全足够。

## 三、一步步操作：从注册到拿到 API Key

### 第一步：注册腾讯云账号

1) 打开浏览器，访问腾讯云官网：[https://cloud.tencent.com](https://cloud.tencent.com)

2) 点击页面右上角的「登录」或「免费注册」按钮。

3) 您可以选择以下方式注册/登录：
   - **微信扫码登录**：使用微信扫描二维码，快速登录（推荐）
   - **手机号注册**：输入手机号，获取验证码，设置密码

4) 按照页面提示完成注册流程。

【建议在此处插入截图：腾讯云官网首页，标注"登录/注册"按钮位置】

### 第二步：完成实名认证

1) 登录成功后，点击页面右上角的用户头像，在弹出的菜单中选择「账号信息」或「实名认证」。

2) 在实名认证页面，您会看到「个人认证」和「企业认证」两个选项。**请选择「个人认证」**。

3) 按照页面提示，填写以下信息：
   - 真实姓名
   - 身份证号码
   - 手机号验证

4) 提交后，系统会自动验证。通常几秒钟到几分钟内即可完成认证。

5) 认证通过后，页面会显示「已实名」状态，并显示认证类型为「个人认证」。

【建议在此处插入截图：实名认证页面，标注"个人认证"选项】

### 第三步：开通文字识别服务

1) 在腾讯云官网顶部导航栏，将鼠标悬停在「产品」上，在弹出的菜单中找到「人工智能」→「文字识别 OCR」。

   或者直接访问：[https://cloud.tencent.com/product/ocr](https://cloud.tencent.com/product/ocr)

2) 进入文字识别产品页面后，点击页面上的「立即使用」或「控制台」按钮。

3) 如果是首次使用，系统会提示您开通服务。阅读服务协议后，点击「立即开通」。

4) 开通成功后，您会自动跳转到文字识别控制台。

【建议在此处插入截图：文字识别产品页面，标注"立即使用"按钮】

### 第四步：查看免费额度

1) 开通服务后，在文字识别控制台左侧菜单，找到「资源包管理」或「免费额度」。

2) 点击进入后，您可以看到已发放的免费资源包：
   - **通用OCR免费资源包**：每月 1,000 次
   - 有效期：当月有效，次月自动刷新

3) 如果没有看到免费资源包，请检查：
   - 是否已完成实名认证
   - 是否已成功开通文字识别服务
   - 尝试刷新页面

【建议在此处插入截图：资源包管理页面，显示免费额度】

### 第五步：获取 API 密钥（SecretId 和 SecretKey）

腾讯云的API调用需要使用「SecretId」和「SecretKey」两个密钥，这与百度云的API Key略有不同。

1) 在腾讯云控制台页面，点击右上角的用户头像。

2) 在下拉菜单中，选择「访问管理」或「API密钥管理」。

   或者直接访问：[https://console.cloud.tencent.com/cam/capi](https://console.cloud.tencent.com/cam/capi)

3) 进入API密钥管理页面后，您会看到密钥列表。如果是首次使用，需要创建密钥。

4) 点击「新建密钥」按钮。

5) 系统会要求您进行安全验证（如短信验证码），按照提示完成验证。

6) 验证通过后，密钥创建成功。您会看到：
   - **SecretId**：密钥ID，用于标识您的身份
   - **SecretKey**：密钥内容，用于签名验证

7) **立即复制并保存**这两个密钥。**SecretKey 只会显示一次**，如果关闭页面后忘记保存，只能重新创建新密钥。

【建议在此处插入截图：API密钥管理页面，标注SecretId和SecretKey位置】

### 第六步：确认密钥权限

1) 创建的API密钥默认拥有该账号下所有服务的访问权限。

2) 如果您需要更精细的权限控制，可以在「访问管理」→「用户」→「用户列表」中创建子用户，并为子用户分配特定权限。

3) 对于普通用户，直接使用主账号密钥即可，无需额外配置。

## 四、拿到 API Key 后，下一步该怎么做

### 如何填入配置

腾讯云OCR使用 SecretId 和 SecretKey 进行身份验证。下面是典型的配置方式：

**Python 示例配置：**
```python
# 将以下信息填入您的配置文件或代码中
TENCENT_SECRET_ID = "您的 SecretId"
TENCENT_SECRET_KEY = "您的 SecretKey"
```

**JSON 配置文件示例：**
```json
{
  "ocr_service": "tencent",
  "secret_id": "您的 SecretId",
  "secret_key": "您的 SecretKey"
}
```

**调用示例（Python SDK）：**
```python
from tencentcloud.common import credential

# 实例化认证对象
cred = credential.Credential(
    "您的 SecretId",
    "您的 SecretKey"
)
```

### 安全保管提示

SecretId 和 SecretKey 拥有您账号的完整操作权限，请务必妥善保管：

1. **绝不要分享给他人**：泄露密钥可能导致他人恶意使用您的云资源，产生高额费用。

2. **不要硬编码在代码中**：建议使用环境变量或配置文件存储密钥，并将配置文件加入 `.gitignore`。

3. **定期更换密钥**：如果怀疑密钥泄露，请立即在控制台删除旧密钥并创建新密钥。

4. **使用子账号密钥**：对于生产环境，建议创建子用户并分配最小必要权限，使用子用户的密钥进行调用。

## 五、常见问题与注意事项

### Q1：免费额度什么时候发放？

**答**：免费额度在您开通文字识别服务后自动发放，无需手动领取。每月初自动刷新 1,000 次额度。例如，如果您在月中开通服务，当月仍可获得完整的 1,000 次额度。

### Q2：免费额度用完后怎么收费？

**答**：免费额度用完后，如果您继续调用接口，会产生费用。计费方式包括：

- **按量后付费**：按实际调用次数计费，通用文字识别约 0.001 元/次起。系统会自动扣费，无需预充值。
- **资源包预付费**：购买调用次数包，价格更优惠。例如 1 万次资源包约 10 元左右。

**建议**：初次使用建议开通按量后付费，这样即使超额也不会产生大额费用。计费详情请参考官方价格页面：[https://buy.cloud.tencent.com/price/ocr](https://buy.cloud.tencent.com/price/ocr)

### Q3：密钥泄露了怎么办？

**答**：如果怀疑 SecretId 或 SecretKey 泄露，请立即采取以下措施：

1. 登录腾讯云控制台
2. 进入「访问管理」→「API密钥管理」
3. 找到泄露的密钥，点击「禁用」或「删除」
4. 创建新的密钥，并更新您的应用程序配置

### Q4：个人认证和企业认证的免费额度有区别吗？

**答**：腾讯云文字识别的免费额度与认证类型无关。无论是个人认证还是企业认证，免费额度都是**每月 1,000 次**。因此，对于普通用户，选择个人认证即可，无需进行企业认证。

### Q5：不同OCR接口共享免费额度吗？

**答**：是的。通用文字识别、卡证文字识别、票据单据识别等大部分接口共享同一个免费资源包（每月 1,000 次）。但部分特殊接口（如增值税发票核验）有独立的免费额度。

### Q6：有地域或网络限制吗？

**答**：腾讯云在国内多个地区部署了服务节点，中国大陆用户访问无特殊限制。腾讯云也提供海外节点，但建议国内用户选择国内地域以获得更好的访问速度。

### Q7：如何查看某个接口是否在免费额度范围内？

**答**：您可以查阅官方文档中的免费额度说明页面：[https://cloud.tencent.com/document/product/866/35945](https://cloud.tencent.com/document/product/866/35945)

该页面列出了各接口的免费额度详情和共享资源包规则。

### 重要注意事项

1. **SecretKey 只显示一次**：创建密钥时请立即复制保存 SecretKey，关闭页面后无法再次查看，只能重新创建密钥。

2. **免费额度不累积**：每月的免费额度不能累积到下月。建议合理规划使用，避免浪费。

3. **QPS 限制**：免费额度有并发调用限制（QPS），如果短时间内大量请求可能会触发限流。如需更高并发，请联系客服申请提升配额。

4. **接口区分**：不同接口的计费单位可能不同，部分接口按「次」计费，部分接口按「页」或「张」计费，使用前请确认。

5. **监控用量**：建议定期查看控制台的用量统计，避免意外超额产生费用。
