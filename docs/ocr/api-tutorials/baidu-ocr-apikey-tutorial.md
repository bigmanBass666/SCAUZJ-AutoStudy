# 百度智能云 文字识别 OCR：如何免费领取 API Key（面向普通用户）

---

**适用对象**：非技术背景的普通用户、个人开发者、学生
**预计耗时**：约 15-30 分钟
**所需材料**：手机号（中国大陆）、身份证（用于实名认证）

---

## 一、这是什么服务

百度智能云文字识别（OCR）是百度推出的一项云端文字识别服务，能够将图片、PDF文档中的文字自动提取出来，转换为可编辑的文本。这项服务支持多种场景，包括通用文字识别、身份证识别、银行卡识别、驾驶证识别、发票识别等数十种专业场景。

**免费额度说明**：百度智能云为已完成实名认证的用户提供免费测试资源。对于通用文字识别等常用接口，**个人认证用户每月可获得 1,000 次免费调用额度**，**企业认证用户每月可获得 2,000 次免费调用额度**。免费额度每月自动刷新，用完后可选择付费继续使用。

**适合谁用**：适合需要在项目中处理图片文字提取的用户，例如：制作证件识别小程序、批量处理扫描件、开发自动化办公工具等。对于个人用户和小型项目，免费额度通常已经足够日常使用。

## 二、开始之前需要准备什么

在开始申请 API Key 之前，请确保您已经准备好以下材料：

### 必须准备

1. **手机号**：需要一个中国大陆手机号，用于注册百度智能云账号并接收验证码。

2. **身份证**：需要进行实名认证，个人用户需要上传身份证正反面照片。这是获取免费额度的必要条件。

### 可选准备

3. **邮箱**：可以绑定邮箱用于接收重要通知和找回密码，但不是必须的。

4. **企业资质**（仅企业认证需要）：如果您选择企业认证，需要准备营业执照等企业资质材料。但**对于普通个人用户，强烈推荐选择个人认证**，流程更简单，免费额度也完全够用。

### 关于银行卡

**不需要绑定银行卡**：百度智能云的文字识别服务使用免费额度时无需绑定银行卡或充值。只有当您超出免费额度并选择付费使用时，才需要进行充值操作。

## 三、一步步操作：从注册到拿到 API Key

### 第一步：注册百度智能云账号

1) 打开浏览器，访问百度智能云官网：[https://cloud.baidu.com](https://cloud.baidu.com)

2) 点击页面右上角的「登录」或「免费注册」按钮。

3) 如果您已有百度账号（如百度网盘账号），可以直接登录；如果没有，选择「注册新账号」。

4) 按照页面提示，输入手机号、获取验证码、设置密码，完成账号注册。

【建议在此处插入截图：百度智能云官网首页，标注"登录/注册"按钮位置】

### 第二步：完成实名认证（获取免费额度的关键步骤）

1) 登录成功后，点击页面右上角的用户头像，在弹出的菜单中选择「实名认证」。

2) 您会看到两种认证方式：「个人认证」和「企业认证」。**对于普通用户，请选择「个人认证」**。

3) 按照页面提示，填写真实姓名、身份证号码。

4) 上传身份证正反面照片。照片需要清晰、完整、无遮挡。

5) 提交后，系统会自动审核，通常几分钟内即可完成认证。认证通过后，页面会显示「已认证」状态。

**重要提示**：只有完成实名认证后，系统才会自动发放免费测试资源。未认证账号无法获取免费额度。

【建议在此处插入截图：实名认证页面，标注"个人认证"选项和上传身份证区域】

### 第三步：开通文字识别服务

1) 认证完成后，在页面顶部导航栏找到「产品」，在弹出的菜单中选择「人工智能」→「文字识别 OCR」。

   或者直接访问文字识别产品页：[https://cloud.baidu.com/product/ocr.html](https://cloud.baidu.com/product/ocr.html)

2) 进入文字识别产品页面后，点击页面上的「立即使用」或「管理控制台」按钮。

3) 如果是首次使用，系统会提示您开通服务。点击「开通服务」按钮。

4) 在开通页面，确认服务条款，点击「立即开通」。

【建议在此处插入截图：文字识别产品页面，标注"立即使用"按钮位置】

### 第四步：创建应用并获取 API Key

1) 开通服务后，您会自动跳转到文字识别控制台。如果没有跳转，可以在页面左侧菜单找到「应用管理」或「应用列表」。

2) 点击页面上的「创建应用」按钮。

3) 在创建应用页面，填写以下信息：
   - **应用名称**：自定义一个名称，例如「我的OCR应用」，方便自己识别即可。
   - **应用描述**：可选填写，简要说明这个应用的用途。
   - **接口选择**：选择您需要使用的OCR接口。对于新手，建议先选择「通用文字识别」等基础接口。

4) 填写完成后，点击「立即创建」按钮。

5) 创建成功后，您会在应用列表中看到刚刚创建的应用。点击应用名称或「查看详情」，进入应用详情页面。

6) 在应用详情页面，您会看到两个关键信息：
   - **API Key**：这是您的应用标识，用于调用接口时身份识别。
   - **Secret Key**：这是您的应用密钥，用于生成访问令牌（Access Token）。

7) 点击 API Key 和 Secret Key 旁边的复制按钮，将这两个密钥复制保存到安全的地方。

【建议在此处插入截图：应用详情页面，标注 API Key 和 Secret Key 的位置】

### 第五步：确认免费额度

1) 在文字识别控制台，点击左侧菜单的「资源包管理」或「免费额度」。

2) 您可以看到各个接口的免费额度使用情况。刚开通服务后，免费额度会在几分钟内自动到账。

3) 如果没有看到免费额度，请检查：
   - 是否已完成实名认证
   - 是否已开通对应的OCR服务
   - 尝试刷新页面或等待几分钟后重试

【建议在此处插入截图：资源包管理页面，显示免费额度状态】

## 四、拿到 API Key 后，下一步该怎么做

### 如何填入配置

拿到 API Key 和 Secret Key 后，您需要将它们配置到您的脚本或应用程序中。下面是一个典型的配置示例：

**Python 示例配置：**
```python
# 将下面的信息填入您的配置文件或代码中
BAIDU_OCR_API_KEY = "您复制的 API Key"
BAIDU_OCR_SECRET_KEY = "您复制的 Secret Key"
```

**JSON 配置文件示例：**
```json
{
  "ocr_service": "baidu",
  "api_key": "您复制的 API Key",
  "secret_key": "您复制的 Secret Key"
}
```

### 安全保管提示

API Key 和 Secret Key 就像是您账号的「钥匙」，请务必妥善保管：

1. **不要分享给他人**：泄露密钥可能导致他人盗用您的额度或产生意外费用。

2. **不要上传到公开代码仓库**：如果您使用 GitHub 等代码托管平台，请确保密钥不会被提交到公开仓库。可以使用 `.gitignore` 文件排除配置文件。

3. **定期更换密钥**：如果怀疑密钥可能泄露，可以在控制台「重置密钥」。

4. **使用环境变量**：建议将密钥存储在环境变量中，而不是直接写在代码里。

### 查看免费额度和用量

1) 登录百度智能云控制台：[https://console.bce.baidu.com](https://console.bce.baidu.com)

2) 进入「文字识别」控制台。

3) 点击左侧菜单的「资源包管理」或「用量统计」。

4) 在这里可以查看：
   - 各接口的免费额度余额
   - 本月已使用次数
   - 历史调用记录

**官方链接**：免费额度查询页面 [https://console.bce.baidu.com/ai/#/ai/ocr/overview/resource/show](https://console.bce.baidu.com/ai/#/ai/ocr/overview/resource/show)

## 五、常见问题与注意事项

### Q1：免费额度什么时候到账？

**答**：免费额度在您完成实名认证并开通文字识别服务后自动发放，通常几分钟内即可到账。如果超过30分钟仍未显示，请尝试刷新控制台页面，或联系百度智能云客服。

### Q2：免费额度用完后怎么收费？

**答**：免费额度用完后，如果您继续调用接口，将会产生费用。您可以选择以下计费方式：
- **按量后付费**：按实际调用次数计费，无需预充值，系统自动扣费。通用文字识别大约 0.002 元/次起。
- **购买资源包**：预付费购买调用次数包，价格比按量付费更优惠。

**建议**：对于初次使用的用户，建议开通按量后付费即可，这样即使超额也不会产生大额费用。计费详情请参考官方价格页面：[https://cloud.baidu.com/doc/OCR/s/tlrzzplc1](https://cloud.baidu.com/doc/OCR/s/tlrzzplc1)

### Q3：密钥泄露了怎么办？

**答**：如果怀疑 API Key 或 Secret Key 泄露，请立即采取以下措施：
1. 登录百度智能云控制台
2. 进入文字识别控制台 → 应用管理
3. 找到对应应用，点击「重置密钥」
4. 更新您的脚本或应用程序中的密钥配置

### Q4：个人认证和企业认证有什么区别？

**答**：主要区别在于免费额度不同：
- **个人认证**：通用文字识别等接口每月 1,000 次免费额度
- **企业认证**：通用文字识别等接口每月 2,000 次免费额度

**对于普通个人用户，强烈推荐选择个人认证**，因为：
1. 流程更简单，只需身份证
2. 企业认证需要营业执照等资质
3. 1000次/月的额度对个人项目已足够使用

### Q5：有地域或网络限制吗？

**答**：百度智能云在国内多个地区部署了服务节点，中国大陆用户访问无特殊限制。如果您在海外，可能访问速度会有所下降，但基本功能正常。

### Q6：免费额度可以累积吗？

**答**：不可以。免费额度每月刷新，未使用完的额度不会累积到下月。建议合理规划使用，避免浪费。

### Q7：如何判断是否超出了免费额度？

**答**：
1. 在控制台的「资源包管理」页面查看余额
2. 调用接口时，如果返回错误码提示「额度不足」，说明已超出
3. 建议在脚本中添加用量统计功能，实时监控

### 重要注意事项

1. **实名认证是获取免费额度的前提**：未完成实名认证的账号无法获得免费测试资源。

2. **不同接口的免费额度不同**：各OCR接口（如身份证识别、发票识别等）有各自的免费额度，请注意区分。

3. **免费额度适用范围**：免费额度仅适用于指定的接口类型，某些高级功能可能不包含在免费额度内，使用前请仔细阅读官方文档。

4. **QPS 限制**：免费额度有并发限制（QPS），如果短时间内大量请求，可能会触发限流。如需更高并发，需要购买 QPS 扩展。
