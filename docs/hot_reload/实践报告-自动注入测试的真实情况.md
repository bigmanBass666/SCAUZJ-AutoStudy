### ⚠️ 常见问题

**Q1: browser_evaluate 报错 "expected string"**
A: 此工具在当前会话不可靠。改用"手动注入"或检查 MCP 配置。

**Q2: 控制台看到 Tampermonkey 的日志混在一起？**
A: 注入前先执行：
```javascript
const oldPanel = document.getElementById('elegant-master-panel');
if (oldPanel) oldPanel.remove();
```

**Q3: GM API undefined？**
A: 确保先注入 `gm-mock.js`，再注入 `main.js`。
