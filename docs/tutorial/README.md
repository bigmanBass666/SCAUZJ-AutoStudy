## 使用建议

1. **从国内服务开始**：如果您在中国大陆，建议先尝试百度智能云或腾讯云，无需绑卡，注册简单。

2. **注意免费额度刷新**：大部分服务的免费额度是按月刷新的，未使用完的额度不会累积。

3. **设置用量监控**：建议在控制台设置预算提醒，避免意外超支。

4. **保护好密钥**：API Key 就像账号密码，请妥善保管，不要泄露给他人或上传到公开仓库。

5. **根据场景选择**：不同服务的 OCR 引擎有不同的优势，根据您的具体需求选择合适的服务。
