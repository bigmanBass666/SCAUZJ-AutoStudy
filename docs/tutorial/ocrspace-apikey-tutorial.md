## 一、这是什么服务

OCR.space 是一家国外公司 a9t9 Software 提供的云端 OCR（光学字符识别）服务。它可以将图片、PDF 文档中的文字内容智能识别并提取出来，返回结构化的文本结果。该服务支持多种语言（包括中文），识别速度快，使用简单。

**免费额度说明**：OCR.space 提供业界非常慷慨的免费额度——**每月 25,000 次免费 API 调用**。这个额度远超大多数同类服务，对于个人项目、学习测试来说完全足够。免费版有以下限制：
- 文件大小限制：1MB（免费版）/ 5MB（Pro版）
- 请求频率限制：每IP每分钟最多500次请求

**适合谁用**：适合需要快速上手 OCR 功能的开发者、学生和研究人员。由于注册流程极简（只需邮箱），无需实名认证，无需绑定银行卡，非常适合想快速体验 OCR 功能的用户。同时，对于需要大量调用的个人项目，25,000次/月的免费额度也非常充足。

> **注意事项**：OCR.space 是境外服务，中国大陆用户访问时可能需要稳定的网络环境。如果访问困难，建议优先使用百度智能云或腾讯云的 OCR 服务。

- **不需要银行卡**：无需绑定信用卡或任何支付方式。

### 关于网络访问

OCR.space 是境外服务，网站地址为 `ocr.space`。中国大陆用户可能需要注意：

- 部分地区可能需要稳定的网络环境才能访问
- API 调用时可能比国内服务稍慢，但基本可接受
- 如果无法访问，建议使用百度智能云或腾讯云作为替代方案

## 三、一步步操作：从注册到拿到 API Key

### 第一步：访问 OCR.space 免费注册页面

1) 打开浏览器，访问 OCR.space 的免费 API Key 注册页面：

   **https://ocr.space/ocrapi/freekey**

   或者在浏览器地址栏输入：`ocr.space/ocrapi/freekey`

2) 等待页面加载完成。如果页面无法打开，请检查网络连接或稍后重试。

### 第二步：填写注册信息

注册页面会显示一个简单的表单，包含以下字段：

1) **Email（邮箱）**：填写您的邮箱地址，这是**必填字段**。API Key 将发送到此邮箱。

2) **Name（姓名）**：可选填写，您可以填写真实姓名或昵称。

3) **其他字段**：页面可能还有其他可选字段，如公司名称、用途说明等，这些都可以不填。

4) 填写完成后，点击页面上的「Register」或「Get Free API Key」按钮。

【建议在此处插入截图：注册表单，标注邮箱输入框和提交按钮】

### 第三步：获取 API Key

1) 提交注册表单后，页面会显示注册成功的提示。

2) 同时，系统会向您填写的邮箱发送一封包含 API Key 的邮件。

3) 打开您的邮箱，找到来自 OCR.space 的邮件（邮件标题类似「Your OCR API Key」）。

4) 在邮件中找到您的 API Key，它通常是一串字母和数字的组合，例如：`K123456789abcdef`。

5) 复制并保存这个 API Key。

**注意**：如果您没有收到邮件，请检查：
- 邮箱的垃圾邮件/垃圾箱文件夹
- 是否填写了正确的邮箱地址
- 尝试重新注册

### 第四步：验证 API Key

1) 您可以通过简单的方式验证 API Key 是否有效。

2) 访问 OCR.space API 文档页面：**https://ocr.space/ocrapi**

3) 在页面上的测试区域，输入您的 API Key，上传一张测试图片，点击测试按钮。

4) 如果返回识别结果，说明 API Key 有效。

## 四、拿到 API Key 后，下一步该怎么做

### 如何填入配置

OCR.space 的 API Key 使用非常简单，只需在请求中添加 `apikey` 参数即可。

**Python 示例配置：**
```python
# 将以下信息填入您的配置文件或代码中
OCR_SPACE_API_KEY = "您的 API Key"
```

**JSON 配置文件示例：**
```json
{
  "ocr_service": "ocrspace",
  "api_key": "您的 API Key"
}
```

**调用示例（Python requests）：**
```python
import requests

url = "https://api.ocr.space/parse/image"
payload = {
    "apikey": "您的 API Key",
    "language": "chs",  # 中文简体
    "isOverlayRequired": False,
}
files = {"file": open("your_image.jpg", "rb")}

response = requests.post(url, data=payload, files=files)
result = response.json()
print(result["ParsedResults"][0]["ParsedText"])
```

### 支持的语言

调用 API 时，可以通过 `language` 参数指定识别语言：
- `chs`：中文简体
- `cht`：中文繁体
- `eng`：英文
- `jpn`：日文
- `kor`：韩文
- `ger`：德文
- 更多语言请参考官方文档

### 安全保管提示

1. **不要公开分享**：虽然 OCR.space 免费版风险较低，但仍建议不要将 API Key 公开发布到 GitHub 等平台。

2. **使用环境变量**：建议将 API Key 存储在环境变量中：
   ```python
   import os
   API_KEY = os.environ.get("OCR_SPACE_API_KEY")
   ```

3. **可以重新申请**：如果 API Key 泄露，您可以随时用新邮箱申请新的 Key。

### 查看用量

免费版目前没有提供用量查看界面，您需要在自己的代码中统计使用次数。建议添加简单的计数功能：

## 五、常见问题与注意事项

### Q1：免费额度什么时候刷新？

**答**：OCR.space 的免费额度按月刷新，每月 1 日重置为 25,000 次。额度不会累积，未使用完的次数不会转到下月。

### Q2：超出免费额度后怎么收费？

**答**：OCR.space 提供 Pro 付费计划：
- **Pro Plan**：约 10 美元/月，包含 100,000 次调用，文件大小限制提升到 5MB
- **Pro PDF Plan**：约 25 美元/月，包含 300,000 次调用，支持更大文件

超出免费额度后，如果您没有升级 Pro 计划，API 调用会返回错误。您可以根据需要选择升级，或等待下月额度刷新。

**官方定价页面**：[https://ocr.space/ocrapi](https://ocr.space/ocrapi)

### Q3：API Key 泄露了怎么办？

**答**：OCR.space 目前不支持自助撤销 API Key。如果 Key 泄露，您可以：
1. 用新邮箱申请新的 API Key
2. 在代码中更新使用新的 Key
3. 联系 OCR.space 官方支持申请作废旧 Key

### Q4：免费版和付费版有什么区别？

| 特性 | 免费版 | Pro 版 |
|------|--------|--------|
| 月调用次数 | 25,000 次 | 100,000 - 300,000 次 |
| 文件大小限制 | 1 MB | 5 MB |
| 请求频率 | 500次/IP/分钟 | 更高限制 |
| 优先级 | 普通 | 高优先级 |

### Q5：在中国大陆能用吗？

**答**：OCR.space 是境外服务，在中国大陆访问可能需要稳定的网络环境。如果您遇到以下情况：
- 注册页面无法打开
- API 调用超时或失败

建议优先使用国内服务（百度智能云、腾讯云）作为替代方案。这些国内服务的网络访问更稳定，且同样提供免费额度。

### Q6：识别精度如何？

**答**：OCR.space 使用先进 OCR 引擎，对印刷体中文、英文识别准确率较高。但对于以下情况可能效果一般：
- 手写文字
- 低分辨率或模糊图片
- 复杂排版或倾斜文字
- 特殊字体

如需更高精度的中文识别，建议使用百度智能云或腾讯云的 OCR 服务，它们对中文场景有更好的优化。

### Q7：支持批量处理吗？

**答**：支持。您可以编写脚本批量调用 API 处理多个文件。但请注意：
- 遵守请求频率限制（免费版每IP每分钟最多500次）
- 统计使用次数，避免超出月度额度

### 重要注意事项

> **⚠️ 网络访问提示**：OCR.space 是境外服务，中国大陆用户可能需要稳定的网络环境才能正常访问和调用 API。如果无法访问，建议优先使用百度智能云或腾讯云 OCR 服务。

1. **文件大小限制**：免费版文件大小上限为 1MB，超过限制的文件会被拒绝处理。

2. **请求频率限制**：免费版每个 IP 地址每分钟最多 500 次请求，超出会返回错误。

3. **无用量监控**：免费版没有提供官方的用量查看界面，建议自行统计调用次数。

4. **不支持撤销 Key**：目前无法自助撤销或重置 API Key，请妥善保管。

5. **邮件可能进垃圾箱**：注册确认邮件可能被邮箱归类为垃圾邮件，请注意查收。
